HOW TO

How to restore MOBIRISE project from your published site.

  1. Create new folder MyRestoredProject in folder C:\Users\<your user name>\AppData\Local\Mobirise.com\Mobirise\projects\
    Create subfolder assets in folder C:\Users\<your user name>\AppData\Local\Mobirise.com\Mobirise\projects\MyRestoredProject\
    Create subfolder images in folder C:\Users\<your user name>\AppData\Local\Mobirise.com\Mobirise\projects\MyRestoredProject\assets\
  2. Open site in Firefox.
    Mouse Right-Click and select View Page Info.
    Click Media tab.
    Click button Select All.
    Click button Save As... and select folder C:\Users\<your user name>\AppData\Local\Mobirise.com\Mobirise\projects\MyRestoredProject\assets\images
  3. In the Firefox address bar, add //project.mobirise at the and of URL
    Save (CTRL-S) project.mobirise into folder C:\Users\<your user name>\AppData\Local\Mobirise.com\Mobirise\projects\MyRestoredProject
  4. Open your saved C:\Users\<your user name>\AppData\Local\Mobirise.com\Mobirise\projects\MyRestoredProject\project.mobirise in Mobirise.

  5. If your site contains more pages, repeat step two for each page.
